## üá∫üá∏ English

### Description
**Dys-Helper** is an inclusive layout and reading assistant designed to improve text accessibility for dyslexic users. Originally developed as a PHP web application, this tool allows dynamic adjustments to typography, spacing, and adds visual aids such as syllable segmentation or a reading ruler.

### Key Features
* **Typographic Customization**: Selection of specialized fonts (OpenDyslexic, Lexend, Atkinson Hyperlegible).
* **Visual Aids**: Colored syllable segmentation, letter confusion highlighting (b/d, p/q), and reading ruler mode.
* **Audio Accessibility**: Text-to-Speech (TTS) with real-time word highlighting (karaoke mode).
* **Profile Management**: Save personalized settings for different students or specific needs.
* **Import/Export**: Support for .docx and .txt files, and configuration export.
